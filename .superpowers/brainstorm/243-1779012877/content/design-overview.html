2. 导航与动画策略

NavHost + AnimatedContent 联合驱动
  • 底部三 Tab → BottomNavBar 切换时触发 AnimatedContent（fade + slide）
  • 右上角菜单 → DropdownMenu + animateFloatAsState 弹性弹出
  • 播放按钮 → animateFloatAsState 控制 scale，按下缩至 0.85，回弹至 1.0 (spring)
  • 背景过渡 → Crossfade 实现绑定图片切换

3. 核心数据流（播放链路）

User 点击播放
  → PlayViewModel.play(audio)
    → PlaybackUseCase.prepareAndPlay(audio)
      → PlayerManager (持有 mainPlayer + bgPlayer)
        → AsmrMediaService (Foreground, 发通知)
          → ExoPlayer.setMediaItem() + play()
        → StateFlow<PlayerUiState> 回流
          → Compose UI collectAsState() 自动重组

5. 关键设计决策

采用

  • 单 Activity + Compose Navigation
  • Foreground Service 保活播放
  • StateFlow + collectAsState 驱动 UI
  • Room + Flow 响应式查询
  • Coil 加载背景图（Compose 原生）

暂不做

  • 云同步（只留接口）
  • Widget（后续迭代）
  • 画中画
  • 音频可视化/均衡器
